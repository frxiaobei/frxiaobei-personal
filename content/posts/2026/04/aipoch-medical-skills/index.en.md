# 450+ Medical Research AI Skills: From Literature Search to SCI Submission

> 450+ medical research AI skills — from literature search to SCI submission, fully covered.

---

## What Is This

**AIPOCH** is an AI skill library built specifically for medical research. It contains 450+ Skills that can be installed directly into AI agent platforms like OpenClaw, Claude Code, and OpenCode.

In short: install these Skills, and your AI assistant becomes a medical research professional.

---

## What It Can Do

The library is organized into four categories:

### 🔍 Evidence Insight

- Search strategy design
- Database selection
- Evidence-level prioritization
- Critical appraisal
- Literature synthesis and gap identification

### 🧪 Protocol Design

- Experimental design generation
- Study type selection
- Causal inference planning
- Statistical power calculation
- Validation strategy

### 📊 Data Analysis

- R/Python bioinformatics code generation
- Statistical modeling
- Data cleaning pipelines
- Machine learning workflows
- Result visualization

### ✍️ Academic Writing

- SCI manuscript drafting
- Methods / Results / Discussion sections
- Meta-analysis narrative
- Cover letters
- Abstract generation

## How to Install

If you use **OpenClaw**, one command:

```bash
bash <(curl -s https://raw.githubusercontent.com/aipoch/medical-research-skills/main/scientific-skills/scripts/openclaw-install.sh)
```

If you use **Claude Code**, just clone into your skills directory.
